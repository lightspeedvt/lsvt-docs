Overview

This page lists all of the settings available for the Boilerplate. There are also examples for common customizations that you can copy/paste into your own projects. Most of the values shown are the defaults.

Global Styles

Site Background
Multiple backgrounds can be specified when using overlays. Note that the stacking order is top down as listed.

$site-bg:( 
    url( img("bg-overlay.png") ) top right no-repeat,
    url( img("bg-site.jpg") ) top center no-repeat
);

Brand Color
Used to highlight certain ui elements.

$ui-brand-color: #3492cf;

Hamburger Menu

Tray Height
These values control the animation when the Hamburger menu is opened and closed. When the user has Quick Links available the admin height is used, alternatively the normal height is used when no Quick Links are available.

$tray-height--admin: 250px;
$tray-height--normal: 190px;

Tray Styles
Any values specified in this "list" are applied to the Hamburger menu's resulting CSS. The most common settings would be to set a repeating background image.

$tray:(
    background: url(img("dark_wall.jpg")),
    background-size: 300px 300px, 
    color:#999
);
