Why Do I Have to Install Stuff?

The boilerplate themes are generated via a specific set of tools that must be installed on your system. Once installed you'll be ready to start generating themes! Once you've installated the tools below, please proceed to the setup instructions.

Basic Installation - Windows, Linux, Etc

The minimum tools you'll need on any system to compile and publish themes are listed below:

  1. GIT - Downloads
  2. SASS - Installation Instructions
  3. NodeJS - Installation Instructions
  4. Grunt CLI - Installation Instructions
  5. Bower - Installation Instructions
